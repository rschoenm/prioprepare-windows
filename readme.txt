Prioprepare is based on a FreePascal Lazarus 
project and uses a SHA256 implementation from 
DCPcrypt (c) 1999-2003 David Barton, modified in 
v2.0.4.1 from Graeme Geldenhuys 2010

-----------------------

DCPcrypt (http://wiki.freepascal.org/DCPcrypt) can 
be made available to Lazarus by: 
1. downloading the latest version (currently 2.0.4.1) from
http://sourceforge.net/projects/lazarus-ccr/files/DCPcrypt/
2. open the lazarus package file "dcpcrypt_laz.lpk"
3. after compilation new components for hashing appear in component palette.

-----------------------

project1.lpr is the lazarus project file.
